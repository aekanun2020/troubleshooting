สำรองไฟล์สคริปต์เวอร์ชันเดิม (ก่อนแก้ปัญหาฝั่ง Windows) — ณ วันที่ 21-07-2026

เหตุผลที่สำรอง: เวอร์ชันนี้ยังมี 2 จุดที่ผู้เรียน Windows ต้องแก้เองด้วยมือ
  (1) มี block ตรวจ container ด้วย findstr /x ที่เด้ง error แม้ container รันอยู่
  (2) ต้อง extract ไฟล์ .gz เองก่อน (python บน host ไม่ทำงาน)

เวอร์ชันในโฟลเดอร์หลักของ repo (นอกโฟลเดอร์นี้) ได้แก้ 2 จุดนี้แล้ว
โดยย้ายการคลาย .gz ไปทำใน container และตัด block ตรวจ container ที่มีปัญหาออก
